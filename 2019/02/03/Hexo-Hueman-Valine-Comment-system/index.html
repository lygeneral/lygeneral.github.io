Hueman添加Valine评论系统

2019-02-03
Hexo, Hueman, LeanCloud, Valine
阅读量次

Hexo的评论系统有很多，常见的有多说、网易云跟帖、畅言、来必力（LiveRe）、Disqus、Hypercomments和Valine，其中多说和网易云已经失效，畅言需要备案，LiveRe加载速度慢。在此，我选择Valine作为博客的评论系统。Valine的详细介绍和使用方法见Valine官网。

LeanCloud配置

首先在LeanCloud官网注册账号，然后进入控制台，创建应用。

然后进入应用后，获取App ID和App Key，后面对Hueman主题进行配置时需要使用。

为了提高数据的安全性绑定域名，进入安全中心在Web安全域名填入域名。

Hueman进行配置

由于Hueman主题提供Valine评论系统的接口，配置Valine评论系统的过程非常简单。在Hueman主题的配置文件_config.yml中找到comment区域，并使用以下参数对valine进行配置，在appId和appKey中分别填入LeanCloud应用中的App ID和App Key。

注意：在添加Valine作为评论系统时，需要将其他几种评论系统删除。由于Hueman主题中comment部分默认使用disqus评论系统，导致我如何设置都不能正常显示Valine，折腾了很久。把disqus后的hexo-theme-hueman删除后，Valine评论系统终于能够正常工作。

评论管理

对博客中的文章进行评论后，进入LeanCloud的控制台中，在存储中找到Class Comment对评论进行管理。
