Java学习之路

2019-11-18
Java
阅读量次

又是真香警告，从大学开始学习了C/C++之后，就固步自封一直使用C和C++编程。听说Java好找工作，也没有学习的动力。时隔至今，终于意识到Java的强大之处，开始了Java的学习之旅。

Java整体的变量、数据结构、整数运算、浮点数运算、布尔运算、字符和数组类型与C语言都非常类似。

Java的字符串String除了是一个引用类型外，还有个重要特点，就是字符串不可变。

占位符 说明
%d 格式化输出整数
%x 格式化输出十六进制整数
%f 格式化输出浮点数
%e 格式化输出科学计数法表示的浮点数
%s 格式化字符串

在方法内部，可以使用一个隐含的变量this，它始终指向当前实例。因此，通过this.field就可以访问当前实例的字段。如果没有命名冲突，可以省略this。但是，如果有局部变量和字段重名，那么局部变量优先级更高，就必须加上this。

一个类只能继承自另一个类，不能从多个类继承。但是，一个类可以实现多个interface
class Student implements Person,Hello{}

因为静态方法属于class而不属于实例，因此，静态方法内部，无法访问this变量，也无法访问实例字段，它只能访问静态字段。
